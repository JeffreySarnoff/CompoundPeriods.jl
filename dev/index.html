Overview

CompoundPeriods

This package enhances the CompoundPeriod type defined within Dates (Dates.CompoundPeriod). A CompoundPeriod is formed by attaching (adding) two or more distinct Periods: Year(1999) + Month(12), Hour(5) + Minute(15) + Second(25).
